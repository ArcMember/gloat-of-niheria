# Роевые
**Роевые** — одни из самых отвратительных ~~по мнению большинства~~ обитателей Нихерии. В отличие от остальных полуразумных существ, эти громадные насекомые, рако- и паукообразные и многоножки нашли себе убежище в глубине лесов и под землёй.

* Практически все существа этой категории **используют местность для охоты**: расставляют ловушки в виде липких луж или прозрачной завесы паутины. Пировать приходят только тогда, когда жертва попалась. Некоторые виды нападают из засады.
* Роевых нередко можно **обмануть**, если ваш запах будет замаскирован.
